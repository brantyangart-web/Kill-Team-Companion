## 项目简介

Kill Team Companion 是一个**战锤40K杀戮小队（Warhammer 40K: Kill Team）**的实体桌游战斗辅助工具。它不是一个电子游戏，而是陪伴玩家在物理桌面上进行对战时的**数字助手**——帮助处理繁琐的骰子判定、伤害计算、回合管理和得分追踪。

### 核心功能

1. **部署选人** — 双方从7名可选特工中选择6名上场，强制包含1名队长
2. **回合管理** — 完整的 Turning Point 流程：先攻判定 → 策略阶段 → 战斗阶段 → 得分结算
3. **射击结算** — 多步引导式射击流程：选目标 → 选武器 → 投骰 → 防御 → 伤害分配
4. **近战搏斗** — 双方同时投骰 → 交替打击/招架选择 → 伤害结算
5. **特殊机制** — 瘟疫守卫的「恶心作呕」DR 5+ 伤害减免、策略卡系统
6. **实时反馈** — 程序化音效、阵亡动画、恶搞台词、实时战况日志
7. **双模式投骰** — 支持随机投骰和物理骰面手动录入
8. **自定义头像** — 支持为特工上传自定义头像图片

### 两个阵营

| 阵营 | 代号 | 模板人数 | 特殊机制 |
|------|------|---------|---------|
| 死亡天使 (Angels of Death) | SM | 7选6 | 爆弹惩戒策略（bolter_discipline） |
| 瘟疫守卫 (Plague Marines) | PM | 7选6 | 恶心作呕 DR 5+、传染韧性策略 |

## 快速开始

```bash
npm install          # 安装 Vite
npm run dev          # 启动开发服务器
npm run build        # 生产构建
npm run demo         # CLI 战斗演示
```
